Requests

Verb Description
GET Used for retrieving resources.
POST Used for creating resources.
PUT Used for changing/replacing resources or collections.
DELETE Used for deleting resources.

Jacked Phones URIs and IDs

Parameter Description Example
Jacked Phones This resource identifier allows you to access the different models that are offered at JP /find/all
Jacked Phones by OS This resource identifier allows you to access the different operating systems that are offered at JP find/os
Add a Jacked Phones This allows you to add a new phone at JP /phones/add
Replace Jacked Phones This resource allows you to swap out phones in our jacked database /replace/model
Delete a Jacked Phones This allows you to delete a phone at JP /delete/model

Responses

All data is received as a JSON object. The Web API Object Model below provides a description of all the retrievable objects.

Phone Object Full

Key Value Description
model string the model of the phone that the object is representing
brand string the brand of the phone that the object is representing
os string the operating system of the phone that the object is representing
image string a URL of the phone that the object is representing sourced from a static image
screensize integer the size of the screen of the phone that the object is representing in inches
